# sleep

## Installation & Update

With cargo:
```bash
cargo install sleep --force
```

## Command Line
```bash
sleep 1.0
Sleep for a given amount of time

USAGE:
    sleep <TIME>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <TIME>    The time to sleep in milliseconds
```

## Examples

Sleep for one second:
```bash
sleep 1000
```
